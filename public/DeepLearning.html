Deep Machine Learning

Reinforcement learning

In some environments Deep Q-Learning models can overestimate Q-values and perform poorly. Double Q-learning solves this by separating the Q-network and target network which reduces overestimation and leads to more stable learning.

The aim is to perform an empirical comparison of Deep Q-Learning and Double Q-Learning models. These models are firstly developed, then evaluated using Taxi-v3 from OpenAI Gym, a reinforcement learning environment where the taxi agent must pick up and drop off customers at the designated locations. The Taxi-v3 environment is tweaked by implementing custom slippery rewards to improve the model accuracy.

Sequence learning

Sequence models interpret and make predictions about any type of sequential data such as audio or text. For the comparison of Sequence Learning algorithms I have developed Recurrent Neural Network (RNN) and Long short-term memory (LSTM) models to train on a dataset of 40,000 Tweets taken from social media. A sequence learning task where the model must accurately predict whether a tweet is labelled in one of the following 3 categories: sadness, neutral, happiness.

Developing CNNs for image classification

I conducted a comparative study into the analysis of Convolutional Neural Network (CNN) models developed to classify images into 3 types of clothing: Casual, Formal and Smart Casual. I firstly developed a simple CNN, then made improvements to produce a 100% accurate CNN.

The first model focuses on a general shape of the image whereas the improved model only focuses on the shape of the clothing item where it is relevant, in this test case it does so with the handbag. The improved model also identifies the model's pose, which it has seen to have a correlation with the category. Models wearing casual clothing change the position of their arms compared to those wearing smart-casual or formal clothing. This visualises how the improved model is learning from the dataset.

Model Architectures

Simple model: For this model I built a CNN with 2 Conv2D layers with ReLU activation followed by MaxPooling2d. A Flatten layer followed, feeding into 2 Dense layers and used softmax for the final activation. This model achieved modest accuracy but struggled with balancing the dataset as the Formal and Smart Casual images were heavily underrepresented.

Improved model: I implemented Transfer Learning into the model and added custom dense layers which greatly helps the model to avoid underfitting. For this I decided to use ResNet50 from Microsoft as it is highly efficient in image classification. Data augmentation techniques allowed me to balance the images used in the train, test and validate datasets by only increasing the number of underrepresented Formal and Smart Casual clothing images. To visualize the testing process, I implemented GRAD-CAM which displayed heat maps showing exactly which features the model was focusing on when deciding to classify an image.
